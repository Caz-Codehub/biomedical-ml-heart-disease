# Biomedical Machine Learning: Heart Disease Prediction

Author: Caden Turnquest


## Project Overview
This project demonstrates how machine learning techniques can be applied to biomedical datasets to predict heart disease using patient health metrics.

The goal is to explore how computational methods and data mining techniques can assist in identifying disease risk patterns in clinical data.


## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn


## Dataset
The dataset contains clinical features such as:

- Age
- Cholesterol levels
- Blood pressure
- Maximum heart rate
- Chest pain type

These attributes are used to predict the presence of heart disease.


## Methods
1. Data preprocessing and cleaning
2. Exploratory data analysis
3. Feature correlation analysis
4. Machine learning classification using Random Forest
